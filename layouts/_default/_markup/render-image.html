{{- /* render-image.html — render hook for images inside article markdown. Every ![alt](src) in a post's body passes through here. If the destination resolves to a processable page-bundle image, it gets the full responsive treatment. Anything else (external URL, non-image, broken ref) falls back to a plain lazy so a bad reference can never break the build. */ -}} {{- $res := "" }} {{- if .Page.Resources }} {{- $res = .Page.Resources.GetMatch .Destination }} {{- end }} {{- if and $res (reflect.IsImageResourceProcessable $res) }} {{- else }} {{- /* External image, non-image, or broken reference — plain lazy img */ -}} {{- end }}
